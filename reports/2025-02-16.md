**Дата:** 16.02.2025

## Код

- **Плагин RoomBroom (ЭДО ↔ iiko):**
  - Восстановлена колонка «Все привязки сделаны?»: красные крестики (✗) снова отображаются для накладных, где не хватает привязок к номенклатуре. Пересчёт привязок (`UpdateAllItemsMappedStatusAsync`) выполняется до отображения грида — галочки и крестики показываются одномоментно.
  - Исправлена ошибка сборки: для обновления грида с фоновой задачи используется `_grid.InvokeRequired` / `_grid.BeginInvoke` (GridControl), а не `_gridView` (у GridView этих методов нет).

- **Упаковка плагина — два ZIP:**
  - В `dist/` формируются два архива: **RoomBroomChainPlugin.zip** (для конечного пользователя, Release с обфускацией) и **RoomBroomChainPlugin-dev.zip** (для разработки, Debug без обфускации). Флаг `-Single` оставлен для сборки одного архива по выбранной конфигурации.
  - Добавлен отдельный архив для пользователя: **RoomBroomChainPlugin-protected.zip** (содержимое из `bin\Release` + конфиг), путь: `edo_iiko_bridge\RoomBroomChainPlugin\dist\RoomBroomChainPlugin-protected.zip`.

- **Безопасность DLL:**
  - Включена многоуровневая защита при сборке Release (ConfuserEx): переименование (preset normal), шифрование строк/констант (constants), ref proxy, control flow, anti tamper, anti debug. Конфиг: `RoomBroomChainPlugin.crproj`.
  - Документация: `RoomBroomChainPlugin/docs/security.md` — описание степеней защиты, двух ZIP, рекомендации при сбое загрузки (отключение anti tamper при необходимости).
