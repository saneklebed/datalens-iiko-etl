# Отчёты и контекст при смене устройства

- **`YYYY-MM-DD.md`** — дневные отчёты (что сделано за день, для руководителя/себя). Примеры: `2026-02-24.md`, `2026-03-04.md`.
- **`context-for-ai.md`** — живой контекст для AI: кратко фиксируем по дням ключевые решения и изменения (Neon, DataLens, доки, плагины), чтобы при открытии проекта с другого ПК (домашний sanek / рабочий Orange) не терять контекст. AI при необходимости читает этот файл для актуальной картины.

**Правило для AI:** при запросе дневного отчёта AI всегда:
1. Создаёт/обновляет файл `reports/YYYY-MM-DD.md` в репозитории.
2. Записывает тот же текст в `C:\Users\Orange\iCloudDrive\Reports\Александр\YYYY-MM-DD.md`.
