# Отчёты и контекст при смене устройства

- **`YYYY-MM-DD.md`** — дневные отчёты (что сделано за день, для руководителя/себя). Пример: `2026-02-24.md`.
- **`context-for-ai.md`** — живой контекст для AI: кратко фиксируем по дням ключевые решения и изменения (Neon, DataLens, доки), чтобы при открытии проекта с другого ПК (домашний sanek / рабочий Orange) не терять контекст. AI при необходимости читает этот файл для актуальной картины.

Дублирование отчётов в iCloudDrive (например `C:\Users\Orange\iCloudDrive\Reports\Александр\`) при необходимости делается вручную или по запросу.
