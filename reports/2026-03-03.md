## Что сделано

### Код
- **Плагин RoomBroom: приход в iiko по кнопке.** Вся цепочка в сборе:
  - Разбор УПД из Диадока: оба формата — UTD (Table/Item) и ФНС 5.03 (ТаблСчФакт/СведТов), перекодировка ответа API (windows-1251) где нужно.
  - Выбор склада: запросы к `api/corporation/stores` и `api/corporation/departments`, парсинг **corporateItemDto** (id, name, type), в комбобокс только STORE/CENTRALSTORE — склады наконец подтягиваются в список.
  - Привязка к номенклатуре iiko: прайс-лист поставщика (GetSupplierPricelistAsync), в XML прихода подставляется `<product>` (guid товара) по коду поставщика.
  - Сборка XML приходной (шапка + строки), POST `documents/import/incomingInvoice`, обработка ошибок (409 «Product not found», 500 по department, IikoImportException с телом ответа).
- Собран ZIP плагина, проверено в iikoChain — приход уходит, склады в комбобоксе есть.
