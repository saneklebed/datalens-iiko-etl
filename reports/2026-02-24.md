## DataLens — дашборд

- **Один период для всего дашборда:** фильтр «Выбор диапазона для анализа»; селектор «Выбор недели» убран. В таблицах везде используется «Период инвентаризации» в формате H5 (например 27.01.–02.02.).
- **Вкладка «Результаты» — четыре ТОП-таблицы:** ТОП по отрицательным отклонениям (%); ТОП по положительным отклонениям (%); ТОП по недостачам ($); ТОП по излишкам ($). Один общий селектор «ТОП N» ([top_n_param]) для всех четырёх.
- **Фильтр по топу расхождений:** для каждой таблицы — поле ранга (по убыванию отклонения) и фильтр в чарте `[поле_ранга] <= INT([top_n_param])`. Реализованы в т.ч. для топа по **положительным** отклонениям в деньгах (поле `[deviation_money_signed]`, ранг `rank_by_surplus_money`) и для топа по **положительным** отклонениям в % (поле `[fact_deviation_pct_qty]`, ранг `rank_by_surplus_pct`). Для положительных таблиц в чарте дополнительно фильтр «только положительные» (например `deviation_money_signed > 0`).
