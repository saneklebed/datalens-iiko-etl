## Код

- **Плагин RoomBroom (ЭДО ↔ iiko):**
  - **Красивые кнопки:** кнопки «Подписать и выгрузить в iiko», «Отказать», «Выгрузить в iiko» оформлены с иконками (✓/✗), цветами (зелёный/красный), серые в disabled; тот же стиль для массовых кнопок (меньше размер). Панель массовых действий скрывается в режиме деталей накладной.
  - **Массовое подписание со своей логикой:** если среди выбранных есть накладная со статусом «внесено в iiko» — массовые кнопки блокируются; статус «накладная в iiko удалена» не считается «уже в iiko», массовые действия для таких с готовыми привязками разрешены. Учёт `AllItemsMapped` для допуска к выгрузке.
  - **Статусность в контрагентах:** в списке контрагентов добавлена отображение статусов (визуальная индикация по данным из ЭДО/интеграции).
  - **Смена даты внутри накладной:** возможность менять дату прихода/документа в карточке накладной.
  - **Приоритезация статуса накладной:** статус накладной в списке и в деталях выводится с заданным приоритетом (какой статус показывать первым при нескольких).
  - Восстановлена колонка «Все привязки сделаны?»: красные крестики (✗) отображаются для накладных, где не хватает привязок; пересчёт до отображения грида — галочки и крестики одномоментно.
  - Исправлена ошибка сборки: обновление грида с фона — через `_grid.InvokeRequired` / `_grid.BeginInvoke` (GridControl).

- **Упаковка плагина — два ZIP:**
  - В `dist/` формируются два архива: **RoomBroomChainPlugin.zip** (для конечного пользователя, Release с обфускацией) и **RoomBroomChainPlugin-dev.zip** (для разработки, Debug без обфускации). Флаг `-Single` для сборки одного архива по выбранной конфигурации.
  - Отдельный архив для пользователя: **RoomBroomChainPlugin-protected.zip** в `edo_iiko_bridge\RoomBroomChainPlugin\dist\`.

- **Безопасность DLL:**
  - Многоуровневая защита при сборке Release (ConfuserEx): переименование (preset normal), шифрование строк/констант (constants), ref proxy, control flow, anti tamper, anti debug. Конфиг: `RoomBroomChainPlugin.crproj`.
  - Документация: `RoomBroomChainPlugin/docs/security.md` — степени защиты, два ZIP, рекомендации при сбое загрузки.
